# دليل المساعدة (Help)

ثلاث شاشات تهدف لتوصيل المستخدم أو أسرته بأقرب جهة مساعدة فعلية، طبية أو نفسية، في أقل وقت ممكن.

---

## 1. Help Map — `help-map.html`

### الغرض (Purpose)
عرض المستشفيات ومراكز علاج الإدمان وجهات الدعم النفسي القريبة من المستخدم، مع أولوية واضحة لزر الاتصال بالطوارئ.

### التخطيط (Layout)
1. Top bar بعنوان وأيقونة بحث.
2. **شريط طوارئ ثابت** (`.emergency-bar`) بخلفية حمراء، يبقى ظاهرًا فوق المحتوى (`sticky`) — أهم عنصر تفاعلي في الشاشة.
3. **تبويبات فلترة** (`.tabs`): الكل، مستشفيات، مراكز علاج، دعم نفسي.
4. **منطقة خريطة** (`.map-placeholder`) — تمثيل بديل (placeholder) لخريطة تفاعلية حقيقية ستُستبدل بمكتبة خرائط (Google Maps / Leaflet) عند الربط الفعلي.
5. **قائمة مراكز** (`.help-list`) بديلة تحت الخريطة، كل عنصر `.center-card` يحتوي أيقونة، اسم، مسافة، وحالة العمل.
6. Bottom navigation مشترك.

### المكونات (Components)
| المكون | الوصف |
|---|---|
| `.emergency-bar` | زر/رابط `tel:` يفتح تطبيق الاتصال مباشرة على رقم الطوارئ |
| `.tab[data-tab]` | فلتر لتصنيف المراكز المعروضة |
| `.center-card` | بطاقة مركز قابلة للضغط تنقل لـ `center-detail.html` |

### المنطق (Logic - `help.js`)
- عند الضغط على تبويب فلترة، يُحدّث الفئة النشطة. منطق الفلترة الفعلي (إخفاء/إظهار حسب التصنيف) هو placeholder حاليًا ويحتاج ربطه بسمة تصنيف حقيقية على كل بطاقة + بيانات API/خريطة فعلية.

### الانتقال (Navigation)
`help-map.html` → `center-detail.html?id=N`

## 2. Center Detail — `center-detail.html`

### الغرض (Purpose)
تزويد المستخدم بكل ما يحتاجه للوصول لمركز معين أو الاتصال به مباشرة.

### التخطيط (Layout)
1. Top bar برجوع.
2. أيقونة المركز + اسمه + تصنيفه (badge).
3. **بطاقة معلومات** (`.center-detail__info-block`) بثلاث صفوف (`.info-row`): العنوان (مع رابط خرائط جوجل)، الهاتف (مع رابط اتصال مباشر `tel:`)، أوقات العمل.
4. وصف نصي عن خدمات المركز.
5. زر بارز "اتصال الآن" (`btn-danger`) في الأسفل.

### المكونات (Components)
| المكون | الوصف |
|---|---|
| `.info-row` | صف معلومة موحد الشكل (أيقونة + عنوان + تفاصيل + رابط فعل) |
| روابط `tel:` و"فتح في خرائط جوجل" | إجراءات مباشرة بدون شاشات وسيطة |

### الانتقال (Navigation)
رجوع فقط لـ `help-map.html`، أو فعل خارجي (اتصال/خرائط).

## 3. Support Corner — `support-corner.html`

### الغرض (Purpose)
ركن داعم نفسيًا لأي شخص يشعر أنه أو أحد أقاربه معرض لخطر الإدمان، بنغمة هادئة وغير تحكمية، مع تأكيد واضح على السرية.

### التخطيط (Layout)
1. Top bar برجوع.
2. **رسالة ترحيبية داعمة** (`.support-welcome`) بخلفية هادئة (تدرج أخضر مزرق فاتح).
3. **خطوط مساعدة** (`.helpline-card`) مقسمة لمجموعتين: "للفرد" و"للأسرة"، كل بطاقة بزر اتصال مباشر.
4. **قسم "كيف تتحدث مع شخص تخاف عليه؟"** بعناصر `<details>` قابلة للطي.
5. **رابط "تواصل بسرية"** (`.anonymous-contact`) بشكل بطاقة بديلة بارزة.
6. ملاحظة خصوصية صغيرة في الأسفل.

### المكونات (Components)
| المكون | الوصف |
|---|---|
| `.helpline-card` | بطاقة خط مساعدة بزر اتصال `tel:` مباشر |
| `.anonymous-contact` | يُفترض ربطه بنموذج تواصل مكتوب لا يُظهر هوية المستخدم لجهة الدعم (شاشة منفصلة عند البناء الكامل) |
| `.privacy-note` | تنويه ثابت بسرية المعلومات، يجب أن يبقى مرئيًا وواضحًا في كل تفاعل بهذا القسم |

### الانتقال (Navigation)
رجوع فقط لما قبلها (عادة `addiction-home.html` أو `home.html`)، أو فعل خارجي (اتصال).
